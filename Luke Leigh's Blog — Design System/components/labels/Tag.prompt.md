Bordered pill for tags/categories (`Tag`) and a solid status label (`Badge`).

```jsx
<Tag href="/tags/powershell" icon="fas fa-tag">PowerShell</Tag>
<Badge variant="success">Published</Badge>
<Badge variant="info">8 min read</Badge>
```

`Tag` is the theme's outlined taxonomy chip; `Badge` is a filled status label on the semantic palette (`primary`/`success`/`warning`/`danger`/`info`/`neutral`).
